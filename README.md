# subp

For finding subdomains of a domain from different public free apis

## Usage

```
subp <domain>
```

## Sources

Please feel free to issue pull requests with new sources! :)

### Implemented
* wayback
* threatminer
* threatcrowd
* alienvault
* dns.bufferover.run
* certspotter.com
* jldc.me
* hackertarget
* urlscan
* sublist3r
* sonar
* crt.sh
